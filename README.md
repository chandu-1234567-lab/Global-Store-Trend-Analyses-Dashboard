# Global-Store-Trend-Analyses-Dashboard
This project uses a retail sales dataset (Global Superstore) containing information such as sales, profit, shipping cost, customer details, order year, region, country, state, and product sub-categories. The dataset spans multiple years and geographical regions, allowing comprehensive analysis of business performance.
